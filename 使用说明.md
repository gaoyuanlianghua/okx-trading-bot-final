# OKX 交易机器人使用说明

## 1. 系统要求

- Windows 10/11 操作系统
- 至少 4GB 内存
- 至少 1GB 可用磁盘空间
- 稳定的网络连接

## 2. 安装步骤

1. 下载并解压 `okx_trading_bot_core.zip` 文件
2. 打开解压后的目录，找到 `okx_trading_bot.exe` 文件
3. 无需安装，直接双击 `okx_trading_bot.exe` 启动应用

## 3. 配置说明

1. 在 `config` 目录中，复制 `.env.example` 文件并重命名为 `.env`
2. 编辑 `.env` 文件，填写 OKX API 相关信息：

```
# OKX API 配置
OKX_API_KEY=your_api_key
OKX_SECRET_KEY=your_secret_key
OKX_PASSPHRASE=your_passphrase
OKX_API_URL=https://www.okx.com
```

3. 保存配置文件后，重新启动应用

## 4. 使用方法

1. 启动应用后，您将看到主界面
2. 在「市场数据」标签页，您可以查看实时市场数据
3. 在「交易」标签页，您可以执行交易操作
4. 在「策略」标签页，您可以设置和管理交易策略
5. 在「风险管理」标签页，您可以设置风险参数

## 5. 常见问题

### 5.1 无法连接到 OKX API
- 检查网络连接是否正常
- 检查 API 密钥是否正确
- 检查 OKX API 服务是否正常

### 5.2 交易失败
- 检查账户余额是否充足
- 检查交易参数是否正确
- 检查网络连接是否稳定

### 5.3 应用崩溃
- 确保系统满足最低要求
- 检查配置文件是否正确
- 尝试重新启动应用
